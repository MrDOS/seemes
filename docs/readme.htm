Seemes 0.2 - Documentation

Introduction

Thanks for downloading Seemes CMS (see-mes see-emm-ess), a content management system designed to make the life of a developer easy.

There are several files in this folder containing documentation. To make maintenance easier, there isn't a menu file on each page. (Although we will add one if all of you promise to install Apache/PHP locally so we can make a dynamic menu! ;) ) However, the filenames are pretty self-explanatory. You really should read installation.htm and basicusage.htm (in that order). Enjoy!

Please note: A fundimental knowledge of HTML is required to operate a website powered by Seemes, and a basic grasp of CSS would be helpful. Keep in mind that the more you know of each, the better your website can be, no matter what (if any) content management system you use.
