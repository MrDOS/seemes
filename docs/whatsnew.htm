Seemes 0.2 Beta 2 - Documentation

What's new

Version 0.2 Beta 2

Release date: 7 June, 2007

Version description

Seemes 0.2 Beta 2 is an update to Seemes 0.2 Beta 1, including bug fixes and feature updates.

New features

  • External links may now be placed in the main menu.
  • Filename entry is now done using a dropdown box instead of a text input box.
  • Menu items now have their own ID's.
  • There is now a page load timer; access the current load time by calling timerreturn().
  • You can now specify how a page should be added to the menu when creating the page.

Bug fixes

  • </textarea>'s no longer terminate the editing space in the Edit page control panel. (This is done by converting all instances of / to the element &#x2F; when the page is loaded for editing; the symbol is automatically reverted to / when the page is saved.)
  • Slashes are now stripped from page names upon page load.
  • The name of the temp variable used to store the content of the menu while loading the Edit menu control panel has been changed from $menu to $menucontent to avoid overwriting the formatted menu. (This bug made menu editing difficult while using theme files that placed the menu after the page content in the page's HTML.)

Version 0.2 Beta 1

Release date: 25 March, 2007

Version description

Seemes 0.2 Beta 1 is a complete rewrite of Seemes 0.1e.

New features

  • Better error reporting.
  • New file management features.

Bug fixes

  • Being a rewrite, no bugs from previous versions have been fixed.
